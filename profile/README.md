# Tejitha Portfolio

This is my personal portfolio website built with **HTML, CSS, and JavaScript**.

## Features
- Responsive navigation bar
- Home/About/Skills/Projects/Contact sections
- Project cards with **Demo Video popup modal**
- Download CV and external links

## Project Structure
- `index.html` - Main page + markup for sections, projects, and video modal
- `style.css` - Styling and responsive layout (including skills grid and video modal)
- `script.js` - Page interactions (menu + demo video modal open/close)
- `assets/` - Images/videos/CV PDF

## How to Run
1. Open `index.html` in a browser.
   - (Recommended) Use a local server so video/assets load properly.

### VS Code (optional)
- Install **Live Server** extension
- Right-click `index.html` → **Open with Live Server**

## GitHub
Push this repository as-is. All assets are referenced locally from the `assets/` folder.

## Notes
- Demo Videos open in a modal overlay (not inline).
- Netlify/Code links are kept unchanged.
